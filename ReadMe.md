# Система учета учеников и курсов

Дипломный проект представляет собой настольное приложение для учета учеников, учебных курсов и записей учеников на выбранные курсы. Приложение разработано на C# с использованием Avalonia UI и локальной базы данных SQLite.

## Описание проекта

Приложение помогает вести простую базу образовательной организации: хранить данные учеников, добавлять учебные курсы, просматривать списки и связывать учеников с курсами. Интерфейс построен в виде одного рабочего окна с таблицами и формами ввода.

Проект подходит для демонстрации базовых навыков разработки desktop-приложения: работа с UI, MVVM-подходом, моделями данных, сервисным слоем и локальной реляционной базой данных.

## Основные возможности

- добавление и удаление учеников;
- хранение контактных данных ученика и родителей;
- добавление и удаление учебных курсов;
- хранение описания, стоимости и длительности курса;
- запись выбранного ученика на выбранный курс;
- удаление ученика с курса;
- просмотр курсов, на которые записан выбранный ученик;
- хранение данных в локальной SQLite-базе.

## Модели данных

В проекте используются три основные сущности:

- `User` - ученик;
- `Course` - учебный курс;
- `UserCourse` - связь между учеником и курсом.

## База данных

Приложение использует локальный файл базы данных:

```text
AvaloniaApplication26/appsss.db
```

При сборке проекта файл базы копируется в выходную директорию приложения. Подключение к базе выполняется через строку:

```text
Data Source=appsss.db
```

## Как запустить проект

1. Установить .NET SDK, совместимый с целевой платформой проекта.
2. Клонировать репозиторий:

```bash
git clone <ссылка-на-репозиторий>
```

3. Перейти в папку проекта:

```bash
cd myproj
```

4. Восстановить зависимости и запустить приложение:

```bash
dotnet run --project AvaloniaApplication26/AvaloniaApplication26.csproj
```

## Сборка

Для проверки проекта можно выполнить:

```bash
dotnet build
```
